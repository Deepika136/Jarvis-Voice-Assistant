# Jarvis-Voice-Assistant
Jarvis Voice Assistant is a Python-based virtual assistant that uses speech recognition and text-to-speech technologies to execute voice commands. It can perform tasks such as Wikipedia search, web browsing, music playback, time announcements, application launching, and email automation.

#Features
1.Voice command recognition
2.Wikipedia search and summary retrieval
3.Open websites such as Google, YouTube, and Stack Overflow
4.Play music from a local directory
5.Announce the current time
6.Launch applications
7.Send emails using voice commands
8.Interactive text-to-speech responses
